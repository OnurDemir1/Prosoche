# Prosoche

Minimal Pomodoro + ambient sound desktop app.

## Installer

- Current version: `1.0.0`
- Installer file: `dist/Prosoche-Setup-1.0.0.exe`
- Build command:

```bash
npm run dist
```

## Run (Development)

```bash
npm install
npm start
```

Optional check:

```bash
npm run check
```

## Privacy

Prosoche stores settings and scenes locally on your device and does not send personal data to cloud services.
