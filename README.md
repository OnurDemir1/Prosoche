# Prosoche

Pomodoro timer with ambient sounds for focused work.

## Installer

- Current version: `1.0.0`
- Setup: `dist/Prosoche-Setup-1.0.0.exe`
- Portable EXE: `dist/Prosoche 1.0.0.exe`
- Build command:

```bash
npm run dist
npm run dist:portable
```

## Run (Development)

```bash
npm install
npm start
```

Optional check:

```bash
npm run check
```

## Privacy

Prosoche stores settings and scenes locally on your device and does not send personal data to cloud services.
